# NFT Case Mini App

This is a Telegram Mini App for animating the opening of NFT gift cases in the GetGive Bot.

## Features
- Simple case opening animation.
- Integration with Telegram Web App API.
- Theme adaptation and haptic feedback.

## Setup
1. Host on Vercel/Netlify/GitHub Pages.
2. Configure in BotFather.

## Files
- index.html: Main page.
- styles.css: Styles and animations.
- app.js: JavaScript logic.
- assets/case.png: Image for the case.
